## Read
#### - GET -
>`/api/amenities/<itemID>`
#### Request
> No Expected Data On Body

#### Success 201
> Returns: 
```json
{
    id: Integer,
    name: String,
    appeal: Integer,
    category: String,
    common: Integer,
    description: String,
    img_url: String,
    amen_id: Integer,
    home_id: Integer,
    included: Integer }
```

#### Failure 400
> Does Nothing
